PROJECT: ALL-IN-ONE LIFE OS (PRODUCTIVITY + FINANCE + JOURNAL)

PURPOSE:
Build a comprehensive personal “Life OS” web application that helps users manage productivity, finances, mental health, and daily planning in one cohesive system.

This is NOT a simple todo app.
This is a private, decision-support system designed for daily use.

=====================================================
GLOBAL REQUIREMENTS
=====================================================

- Use Next.js (App Router preferred)
- Clean, modern, calm UI (dark/light mode)
- Responsive (desktop-first, mobile-friendly)
- Email/password authentication
- Multi-user support with strict per-user data isolation
- Persist all data per user in database
- Add TODO comments if any part is simplified for performance

=====================================================
GLOBAL USER SETTINGS (MUST PERSIST)
=====================================================

Create a Settings module that persists per user:

Settings fields:
- currencyCode (default: PHP)
- locale (default: en-PH)
- theme (light | dark)
- payoutScheduleType:
  - monthly
  - semiMonthly
  - custom
- monthlyPaydayDay (number)
- semiMonthlyDays (array, e.g. [1,15])
- customPaydays (array, e.g. [5,20])
- workingDaysConfig:
  - working
  - rest
  - sickLeave
  - annualLeave
  - customLeave (custom label + color)

REQUIREMENTS:
- Settings must save and load correctly
- Currency must instantly reflect across the entire app
- Use Intl.NumberFormat everywhere (NO hardcoded currency)
- Payday countdown must use these settings globally
